## Purpose

Master brief Section 12 (Module 12, lessons 106 and 109) requires that
time-sensitive, currently-sold-model recommendations live in a **separate,
dated file** rather than inside the evergreen lesson content in
`src/content/computer-rebuild/lessons.ts`. This is that file.

The lessons themselves teach the *comparison method* (how to read a
processor name, how to evaluate an Apple Silicon generation, how to weigh
condition vs. spec) — durable content that doesn't need to change when a
new chip ships. This file is where specific, dated, market-current
recommendations go once someone has actually done that research.

## Rule for updating this file

- Every entry below must carry the date it was verified.
- Do not let the evergreen lessons reference specific current prices,
  specific currently-sold models, or a ranked "best chip" list — that
  content belongs here, not there.
- When this file is updated, bump the "Last verified" date at the top.
- Prices and availability are highly local (Haiti import costs, USD/HTG
  exchange rate, seller markup) — do not present a single global price as
  authoritative.
