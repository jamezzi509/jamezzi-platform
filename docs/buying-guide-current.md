## Purpose

The buying module needs time-sensitive recommendations to live separately from
evergreen lessons. The lessons teach a durable comparison method. This internal
worksheet is reserved for specific models, observed prices, availability, and
local buying considerations after they have been researched.

## Publication rule

Do not link or quote this file in a student-facing lesson until every published
claim has current evidence and a named reviewer approves it.

Every recommendation must include:

- exact model or model family;
- source URL;
- seller and market location;
- observed price and currency;
- verification date;
- warranty and return-policy notes;
- repairability or parts-availability notes;
- the learner need the recommendation serves.

Prices and availability vary by location, exchange rate, import costs, and
seller markup. Never present one price as universally authoritative.

## Official support facts used in Module 3

Verified 2026-07-20. Recheck before revising or translating the lesson.

- Microsoft Windows 11 requirements: 4 GB RAM and 64 GB storage are minimum
  eligibility requirements, not Jamezzi's comfort recommendation.
  <https://www.microsoft.com/windows/windows-11-specifications>
- Microsoft Windows 10 support ended October 14, 2025. A one-year Consumer
  Extended Security Updates option may apply to eligible devices, but a normal
  buying recommendation should verify Windows 11 eligibility on the exact PC.
  <https://support.microsoft.com/windows/windows-10-support-ends-on-october-14-2025-2ca8b313-1946-43d3-b55c-2b95b107f281>
- Apple macOS Tahoe 26 compatibility list. Exact-model verification is required;
  Apple-silicon MacBook Air models from 2020 onward are listed, while only a
  limited set of Intel Mac models remain.
  <https://support.apple.com/122867>
- Apple Activation Lock for Mac. The previous owner must remove Activation Lock
  before a used Mac changes ownership.
  <https://support.apple.com/102541>

The 16 GB RAM and 512 GB SSD student baseline is an editorial buying target for
comfortable general use, not an operating-system requirement and not a promise
that every application will run. Required software always takes priority.
